# DiscordBot Scaffold

## About

This is just a quick boilerplate discord bot i made so that people can get started 
with learning how to make them and study the code

## How to use

- Install Git (https://git-scm.com/downloads).

- Install NodeJS (https://nodejs.org/en/download/)

- Clone project repository ```git clone https://github.com/grafitely/DiscordBot-Scaffold```

- Open a command prompt window in project directory.

- Run ```npm i discord.js dotenv```.

- Fill in required .env values

- Run ```node index.js```
